Tibetan Wylie Markdown Editor

Learning to read classical and standard modern Tibetan is difficult enough let alone learning how to write in Uchen script. Creating Tibetan based written documents is even more tricky for the novice without a specialised keyboard and/or software. This simple online markdown editor is designed to produce Tibetan text in the Uchen script with ease. At present simple plain text files can be uploaded or saved with support coming soon for integration with GitHub.
